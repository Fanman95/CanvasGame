Game:

1. Press "P" to pause the game

2. Press other key to continue

3. Get the highest point as you can

4. You can get skill point per 5000 score, max. 50 points

5. Press "Q" "W" "E" "R" to release skills

6.
"Q": 30s CD
"W": 60s CD
"E": 30s CD
"R": 120s CD

7. Press F11 to play this game with fullscreen
